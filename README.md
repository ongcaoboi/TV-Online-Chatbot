## Chatbot

 - SỬ dụng thư viện nlp.js 
 - Sử dụng facebook messenger làm giao diện người dùng
 - Tạo api chat để có thể phát triển thêm nhiều giao diện người dùng
 - Tạo giao diện nhắn tin trên nền tảng web gọi tới api chat
